概要

クロスプラットフォームで動作する汎用掲示板クライアント

ダウンロード

Windows
以下のリンクから、win32/win64のzipファイルをダウンロードしてください
Linux
Debian/Ubuntu
コマンドラインから、以下を実行し /etc/apt/sources.list にレポジトリの場所を追加してください。
# echo "deb http://dl.bintray.com/hiroyuki-nagata/deb /" | sudo tee -a /etc/apt/sources.list 
# echo "deb-src http://dl.bintray.com/hiroyuki-nagata/deb /" | sudo tee -a /etc/apt/sources.list
apt-get コマンドにてxrossboardがインストール可能であることを確認してインストールしてください。
# apt-get update
# apt-cache search xrossboard
xrossboard - Cross Platform 2ch browser
# apt-get install xrossboard
Mac OS X
以下のリンクから、dmgファイルをダウンロードしてください

※実行できない時は、「システム環境設定 -> セキュリティとプライバシー」からダウンロードしたアプリケーションの実行許可を与えてください

ビルド

詳細はINSTALLファイルを見てください。

  $ ./bootstrap  
  $ ./configure  
  $ make
